Projeto para gerenciamento de livros.
Autor: Bruno Ferri

Modo de usar:
A página inicial fornece ao usuário as opções "Visualizar Livros", "Cadastrar Livros", "Cadastrar Pessoas", "Cadastrar Empréstimos", "Visualizar Empréstimos" e "Gerenciar Categorias".

Visualizar Livros:
O usuário é direcionado para uma tela na qual poderá ver as informações dos livros, podendo e excluir e editar os registros dos livros conforme a necessidade.

Cadastrar Livros:
O usuário é direcionado para uma tela de cadastro onde poderá registrar o livro no sistema.

Cadastrar Pessoas:
O usuário é direcionado para uma tela de cadastro de usuário onde será possível registrar os dados da pessoa que irá usufruir de empréstimos de livros.

Cadastrar Empréstimos:
O usuário é direcionado para uma área onde deverá registrar o ID do usuário que está pegando um livro emprestado e o ID do livro que está sendo emprestado. Somente é possível registrar um empréstimo se os registros de usuário e livro constarem no sistema. Para cada livro emprestado é necessário um registro de empréstimo.

Visualizar Empréstimos:
O usuário é direcionado para uma tela onde pode visualizar os empréstimos feitos, bem como editá-los e excluí-los.

Gerenciar Categorias:
O usuário é direcionado para uma tela onde poderá ver as categorias existentes, excluí-las, editá-las bem como criar novas categorias.
